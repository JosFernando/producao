# Plataforma de Gestão de Resíduos

This is a single-page application for managing waste, refactored for clarity and maintainability.

## Configuration

To run this application, you need to configure your Firebase API key.

1.  Open the `js/firebase-config.js` file.
2.  Replace the placeholder `__REPLACE_WITH_YOUR_API_KEY__` with your actual Firebase API key.

## Running the Application

Because this application uses JavaScript modules, it must be run from a local web server.

1.  Navigate to the root directory of the project in your terminal.
2.  Start a simple Python web server with the following command:

    ```bash
    python -m http.server 8000
    ```

3.  Open your web browser and navigate to `http://localhost:8000`.
