# Plataforma de Gestão de Resíduos

This is a single-page application for managing waste, refactored for clarity and maintainability.

## Configuration

To run this application, you need to configure your Firebase API key.

1.  Open the `js/firebase-config.js` file.
2.  Replace the placeholder `__REPLACE_WITH_YOUR_API_KEY__` with your actual Firebase API key.

## Verification

The `verify_refactor.py` script and `verification.png` are part of the development and verification process and can be ignored.
